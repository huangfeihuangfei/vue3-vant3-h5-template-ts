## 项目预览地址：https://quliangen.github.io/vue3-vant3-h5-template-ts/ (h5项目请自行f12)
## 集成技术栈：
1. vue3 + vue-router4 + vuex4
2. vant-ui3(按需import组件)
3. pug
4. less
5. h5 px 2 viewPort
6. eslint + prettier （风格校验 + 一键修复）
7. travis CI push代码自行构建gh-pages

## Feature:
1. axios
2. h5示例页面
